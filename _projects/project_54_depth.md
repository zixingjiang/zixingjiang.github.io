abstract: This is a two-person team course project for <i>ECE4513 Image Processing and Computer Vision, Fall 2022</i> at CUHK-Shenzhen. In this project, we derived a single-camera motion stereo algorithm that uses the parallax of an object in two consecutive frames to estimate the depth of an object relative to the camera. Furthermore, we integrated the acquired visual information into a robot odometry. MATLAB simulation verified that the algorithm can reduce the cumulative error of the traditional odometry.
